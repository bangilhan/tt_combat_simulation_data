# 브라우저 콘솔 확인 방법

## 1. 파일 열기
- `index_focused_aimbot_analysis.html` 파일을 브라우저로 드래그 앤 드롭
- 또는 파일 탐색기에서 파일을 우클릭 → "연결 프로그램" → Chrome/Edge 선택

## 2. 개발자 도구 열기
- **F12** 키 누르기
- 또는 우클릭 → "검사" / "Inspect"
- 또는 **Ctrl+Shift+I** (Windows) / **Cmd+Option+I** (Mac)

## 3. 콘솔 탭 확인
- 개발자 도구가 열리면 상단의 **"Console"** 탭 클릭
- 빨간색 에러 메시지 확인

## 4. 확인할 명령어 입력
콘솔 하단의 입력창에 아래 명령어를 하나씩 입력하고 Enter:

### 4.1. 에러 확인
```
에러 메시지가 빨간색으로 표시되어 있을 것입니다.
예: "Uncaught SyntaxError: Unexpected token '{'"
```

### 4.2. window.simulation 객체 확인
```
window.simulation
```
- 결과가 `undefined`면 객체가 생성되지 않은 것입니다
- 결과가 `CombatSimulation {...}` 형태면 정상입니다

### 4.3. markdownToHTML 함수 확인
```
typeof window.simulation.markdownToHTML
```
- 결과가 `"function"`이면 정상입니다
- 결과가 `"undefined"`면 함수가 정의되지 않은 것입니다

### 4.4. renderDashboard 함수 확인
```
typeof window.simulation.renderDashboard
```
- 결과가 `"function"`이면 정상입니다
- 결과가 `"undefined"`면 함수가 정의되지 않은 것입니다

### 4.5. 클래스 구조 확인
```
window.simulation.constructor.name
```
- 결과가 `"CombatSimulation"`이면 정상입니다

## 5. 스크린샷 찍기
- 콘솔의 에러 메시지를 스크린샷으로 찍어서 보여주시면 더 정확히 도와드릴 수 있습니다
- 또는 에러 메시지를 복사해서 알려주세요
